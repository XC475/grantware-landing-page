# GrantWare AI Landing Page

AI-powered grant discovery and management platform.

**Live Site:** [https://www.grantware.ai](https://www.grantware.ai)

---

## Project

The production landing page is in the `grantware-landing-page-final/` directory.

See [grantware-landing-page-final/README.md](./grantware-landing-page-final/README.md) for full documentation.

---

## Quick Start

```bash
cd grantware-landing-page-final
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

---

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4
- **Animation:** Framer Motion
- **Analytics:** Google Analytics 4, Vercel Analytics
- **Deployment:** Vercel
